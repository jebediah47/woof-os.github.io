Woof OS

Woof OS is a simple Arch based distribution for users who want a quick Arch installation with a comfortable, functional eye candy experience.

Features

A beautiful desktop experience.

Woof OS has the latest config files and the team always tries to make the best looking desktop experience.

Beautiful & lightweight

Woof OS has been made with looks and small resource consumption in mind from the ground up, with each release Woof OS config files are optimized and improved for the best UX.

Free and Open-Source

Every single part of Woof OS is 100% FOSS, every ISO profile is licensed under the GNU General Public License v3.0, utilities and websites are licensed under the MIT license.

It's super fast

Due to Woof OS being based on Arch Linux, that makes it super fast accordingly! The combination of the Arch base and lightweight DEs/WMs makes for a snappy OS.

User friendly

Woof OS uses the Calamares graphical installer which makes installing it super easy! But power users have not been left aside! We offer both the easier to use XFCE desktop and a Qtile WM edition.
